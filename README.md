# 🛠 How To Use The Website Properly

## Step 1 — Open the Website

Open the HTML file in your browser.

Recommended browsers:

* Google Chrome
* Microsoft Edge

---

## Step 2 — Generate Mock Test JSON

1. Click **"Copy AI Prompt"**
2. Open an AI model like:

   * ChatGPT
   * Claude
   * Gemini
3. Paste the copied prompt
4. Upload your study materials
5. Ask the AI to generate IMUCET JSON

---

## Step 3 — Paste JSON

1. Copy the generated JSON
2. Paste it inside:

   ```json
   Paste Exam JSON
   ```
3. Click:

   ```
   Validate & Initialize Test
   ```

---

## Step 4 — Candidate Login

1. Enter your full name
2. Generate Hall Ticket
3. Continue to Instructions

---

## Step 5 — Start Exam

* Read instructions carefully
* Accept terms & conditions
* Start the exam

⚠ Important:

* Do NOT switch tabs
* Do NOT exit fullscreen
* Doing so may auto-submit the exam

---

## Step 6 — Attempt Questions

Available actions:

* Save & Next
* Mark for Review
* Previous Question
* Clear Response

Question Palette Colors:

* 🔵 Not Answered
* 🟢 Answered
* 🔴 Marked for Review
* 🟡 Answered & Marked

---

## Step 7 — Submit Test

After completion:

* Submit the exam
* Generate Premium Report

---

## Step 8 — Analyze Performance

The system provides:

* Accuracy Analysis
* Subject-wise Analytics
* Weak Areas
* Wrong Questions
* Review Questions
* Performance Index
* PDF Report Download
